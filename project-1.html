A quiz app that tests a user's knowledge of various topics. This app was created using ReactJS.

Project Overview

The work for this project was based on a provided Figma design. The plannig of the app involved structuring app into various resuable components. Durin the coding process a systematic approach was followed to insure that the logic of the game functioned as expected.

Learning to use the useEffect hook was a key part of this project. This is because it was not only used to fetch data from the Open Trivia DB api but also at other stages of the game when aspects of the state. In its current state the app has good functionality but there is a lot of room for improvement.
